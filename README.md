# Retail-Sales-Analytics-Dashboard

## Project Overview

This Power BI dashboard provides insights into retail sales performance, profitability, returns, and regional trends. The dashboard helps businesses monitor key performance indicators (KPIs), identify top-performing products, analyze return rates, and evaluate profit margins across different regions.

## DAX Measures Used

```DAX
Total Sales = SUM(Orders[Sales])

Total Profit = SUM(Orders[Profit])

Profit Margin % =
DIVIDE([Total Profit],[Total Sales],0)

Return Rate % =
DIVIDE(SUM(Orders[Returns]),[Total Sales],0)
```
